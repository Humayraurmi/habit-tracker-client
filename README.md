# Freelance Marketplace Platform

*Live Site:* https://shiny-fenglisu-57ffcb.netlify.app/
*Backend API:* https://habit-tracker-server-seven.vercel.app/
A responsive habit tracker application where users can add habits, browse public habits. The platform supports secure authentication, real-time habits update, and personal habit management features.

## Features

- *User Authentication:* Secure login and registration using Firebase Authentication (Email/Password & Google Sign-In).
- *Add and Manage habit:* Users can add new habits with details like title,category, importance.
- *Responsive UI:* Fully mobile-friendly navigation with profile and logout dropdown visible across screen sizes.
- *Secured Backend:* All API routes are protected using MongoDB Atlas.

---

## Technologies Used

*Frontend:*
- React.js
- React Router
- Firebase Authentication
- Tailwind CSS
- React Hot Toast / React Toastify/ sweet alert

*Backend:*
- Node.js
- Express.js
- MongoDB Atlas
